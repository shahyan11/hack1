My Personal Library

Core Idea: Accessibility, imagine wanting to use the school library on a Saturday or closing hours. If the library is closed there’s no way to use it. So with the virtual library we can access the school’s library from home with the use of our student info/access information. That’s the core of our product, easy accessibility.
